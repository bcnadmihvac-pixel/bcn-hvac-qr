# BCN HVAC · QR (GitHub Pages)

## Pasos
1. Subí todo este folder al repo **bcn-hvac-qr** en GitHub (o cambiá el `base` en vite.config.ts si usás otro nombre).
2. En Settings → Pages, seleccioná **Source: GitHub Actions**.
3. Hacé un commit; el workflow **Deploy Vite to GitHub Pages** corre solo.
4. Tu URL final será: `https://TU-USUARIO.github.io/bcn-hvac-qr/`
